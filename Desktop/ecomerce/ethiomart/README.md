# Ethiomart

Monorepo scaffold for Ethiomart (MERN marketplace)

## Quick onboarding

- Install root deps (optional workspace management):

```powershell
cd ethiomart
npm install
```

- Run backend in dev:

```powershell
npm run dev:backend
```

- Run client (from `client` folder) after installing its deps:

```powershell
cd client
npm install
npm run dev
```

Contributions: follow the folder conventions: `backend`, `client`, `seller`, `admin`, `modules`.
